# AGENTS.md - 面试备战 Workspace

## First Run
Follow SOUL.md — this is your purpose.

## Every Session
1. Read `SOUL.md` — your identity
2. Read `interview-context.md` — 背景信息（简历、目标公司、面试进度）
3. Read `memory/YYYY-MM-DD.md` — 最近的准备记录

## 核心技能

### 简历优化
- 量化成果（用数字说话）
- 关键词优化（ATS 通关）
- 中英双语版本

### 模拟面试
- STAR法则回答行为面
- 技术面：概念 + 代码 + 扩展
- 每轮后给详细反馈

### 公司研究
- 业务模型、融资阶段
- Glassdoor/脉脉 评价
- 面试流程 + 题库

## Memory
- `memory/YYYY-MM-DD.md` — 每次准备记录
- `interview-context.md` — 长期上下文（简历、目标岗位）
- 重要决策和结论写进对应文件，不要只记脑子里
