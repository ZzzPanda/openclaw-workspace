# CanIRun.ai — 本地 AI 模型兼容性检测

**URL**: https://www.canirun.ai/

## 是什么

浏览器内硬件检测工具，100% 本地运行（不传数据到服务器），分析你的 GPU/CPU/RAM 后告诉你能跑哪些 AI 模型。

## 核心功能

- **模型兼容性检测** — 输入模型名，返回能否跑 + 预估速度
- **GPU 对比** — 买新卡前横向比较两个 GPU 跑模型的差距
- **Tier List** — 你的硬件在同类中排什么档次，可分享

## 常用场景

用户想跑本地大模型但不知道硬件够不够 → 直接甩链接

## 常用模型对照（大概门槛）

| 硬件 | 能跑什么 |
|------|---------|
| RTX 3060 12GB | SDXL ✅ / 70B Q4 ❌ |
| RTX 4090 24GB | SDXL ✅ / 405B ❌ |
| RTX 5080 16GB | SDXL ✅ / 405B ❌ |
| Mac M3 24GB | SDXL ✅ / 70B Q4 ❌ |

> RTX 5080 + 32GB RAM 基本上除了最大的闭源模型都能跑
