# 2026-03-13 补充

### Slay The Robot APK 打包
- 打包命令：`godot --headless --export-debug "Android" builds/android_debug.apk`
- **重要**：文件名必须带时间戳，例如 `android_debug_20260313_1626.apk`
- 同步到 Google Drive：`/Users/roger/Google 云端硬盘/apk/Slay-The-Robot/`
- 打包前确保 export_presets.cfg 配置正确

### 当前问题
- Godot 4.6.1 命令行打包报错："Cannot export project with preset 'Android' due to configuration errors"
- 需要手动在 Godot Editor 中配置 Android Export 后才能命令行打包
