## 修复的 Bug

### 1. CombatRangeIndicator 显示位置修复
**问题**：战斗范围指示器使用 `Global.player_data.player_position_x` 获取玩家位置，但该值可能未同步。

**修复**：
- 改为从玩家节点直接获取 `position_x`（逻辑坐标）
- 添加与 `BaseCombatant` 一致的坐标转换：`screen_x = COMBAT_MARGIN + (logic_x / COMBAT_WIDTH) * combat_area_width`
- 修复 `_draw()` 中的范围绘制逻辑

### 2. ActionRecoil 后坐力方向计算修复
**问题**：当 `attack_direction=0`（默认值）时，`sign(0)=0` 导致后坐力为 0。

**修复**：
- `attack_direction=0` 时默认为 `1.0`（向右攻击）
- 使用 `-sign()` 计算后坐力方向，与攻击方向相反

### 3. NewRunMenu 初始神器显示优化
**问题**：TODO 注释指出只能显示一个初始神器。

**修复**：
- 如果有多个初始神器，在描述前添加 `[+N 神器]` 提示
- 保留第一个神器的完整描述
