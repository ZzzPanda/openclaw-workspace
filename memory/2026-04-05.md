### OpenClaw Skill 开发
- 用户想在 Control UI 里添加 `/local-lm` slash 命令，直接调用本地 LM Studio
- 创建了 `local-lm` skill：
  - 路径：`~/.openclaw/workspace/skills/local-lm/`
  - 脚本：`scripts/call_lmstudio.py`（已测试 ✅）
  - LM Studio 地址：`http://192.168.30.16:1234/v1`
  - 模型：`mistralai/ministral-3-14b-reasoning`
- 自定义 `/local-lm` slash 命令无法注册进 Control UI（slash 命令系统为内置保留）
- 最终方案：
  - 自然语言触发："用 local-lm 回答：..."
  - 文本命令：`!local-lm <提示词>`（作为 fallback）
- bash exec 需要 openclaw 配置 `commands.bash=true`（Control UI 需要在配置里开启）
