# 5、开盘定式


## 一、开盘定式（前期基本功）：T+0是在日线框架范围里的，找上涨趋势中的波动率大的热点强势股，找急涨（卖）急跌（买），大钱都输在确定性（市场没有确定性），预则立不预则废，不在盘中临时参与一个品种，只做自己熟悉的模式，

- 要有交易边界（竖起来的第一次分歧加仓、第二次分歧不接、第三次分歧清仓）（50W账户占3%）做好完善自己的交易体系（识势、懂势、借势）
- 1、单K 压力测试：①回调看左边阳线的支撑。②反弹看左边阴线的压力。(单K线有五个价格即最高开盘收盘实体一半最低，强弱分水岭在实体的阴半阳半，每日预判)。③K线越近力道越大，K线越大力道越大，K线越多支撑压力越多
- 2、结构压力测试：①下降通道中的第一根阴线的最高实体压力重，没有突破不能全面做多。下跌趋势中的每一个整理区间的阴线最高实体都是压力。
- ②上升通道中每一个最低阳线（未创左低）的实体最高价被突破，做多低吸高抛做正T。
- ③横盘震荡中在区间做T，上轨反T下轨正T
- 3、趋势拐点：趋势下跌的最低阳线的实体的最高价被突破趋势拐点，最低阳线的最低价为止跌关键，做正T（低部必然是下影线或反包线）

## 二、分时图开盘定式（涨停板坐标）以昨天的收盘价0为中心上下各七格，上为对岸下为本岸，开盘在正负一格之内时是震荡区不买股，无法做T。前30F走势决定一天走势，尾盘30F走势是明天30F缩影

- 1、高开高走开盘定式：①强势跳空缺口高开1.43或在8.56超强开盘会一波拉涨停，开在对岸为强势开盘做涨势正T，拉回（分歧）不能跌破开盘价，有窗胜无窗，MACD须一波红丰满型（绿的诱多）高开高走一波红量能要跟上，高开高走回踩买，先买再卖做正T。
- ②强势股走331模式（2221模式弱势不做，只做急涨急跌波动率大的）即放量突破一格后下一个压力在三格上（在此再增量就易放量涨停，三个放量点要依次递增不能背离），若股价大于均价线三格若量价背离绝对卖或没有顶背离的MACD死叉也离场，突破量价不能背离否则涨停都要卖，超强股应一波拉涨停否则诱多
- 2、平开高走开盘定式：过左高又过对岸一（N字结构，放量拉升回踩买分歧），最好不能跌破均价线，过对岸一1.43低吸上车，通常放量突破对岸一后至少涨二格强的涨三格（1.43*3），做正T，买分歧不能跌破均价线，MACD鳄鱼嘴离场
- 3、低开低走开盘定式：①开在本岸一格及以下集合竞价9：24分直接跑准备做倒T，MACD空中漏油，会跌弱两强三格（再伺机买回--均价线不随股价下跌）,若直接低开在三格就没有T的机会了
- ②低开低走拉拐卖（如果集合竞价没有来得及卖的，反弹负一上或第一次反抽均价线卖），有缺胜无缺，只要开盘价不突破都会创新低（若该跌不跌理应看涨突破本岸一回踩买等三格）
### 开盘三定位

- ②9:30F至9:45F的最高和最低形成的自然箱体是一天的强弱分水岭（第一根15FK线），上下翻箱体（阳上阳正T或阴下阴倒T）三天看趋势七天看箱体

## 三、开盘价为多空分水岭（如果开盘价被跌破一定要走两横一竖），掌握以下三种形态（开盘定式决定T+0方向正T或倒T）（共有八种开盘搞懂这三种包含一切就够用了）

- 1、高开高走（开盘价三种状态之一）：向上跳空缺口，这种开盘的两种意义趋势上攻和拉高出货，
- ①趋势上攻（高开高走等分歧回拉最好在均价线之上且MACD鸭嘴张嘴两波红高走N字不破从开盘价计算的旗杆的一半--量缩分歧结束低吸且买点须在均价线上，破低过高回拉和过高回拉再过高（拉到一三格有安全垫）两种分歧，多空搏杀买在分歧，放量转一致MACD开口陡角启动）实战中最稳定。
- ②高开高走拉高出货（冲高后跌破开盘价拉拐卖，分歧点直接破开盘价。杀跌也是分歧但空方力道大于多方看空。受到昨天金色2：30买入人的抛压，今天冲高买的全部被套，所以一定不要第一波拉升就买票等分歧后再做）
- ③高开高走一波拉涨停凤尾型（一波拉出旗杆后走两竖一横整理，整理低点不破时，放量一波拉涨停）。高开高走最佳模式：旗杆拉升三四格，最差回调一格
- 2、低开震荡高走：向下跳空缺口，这种开盘的两种意义洗盘上攻和强行出货。低走看开盘级别，开在本岸三格之下，打拐直接卖。
- ①洗盘上攻（一放二缩三突破等三格卖）：低开低走走出双（多重）底结构，两低一高走完洗盘上攻（等旗杆后的分歧），两横一竖指当日股价突破均价线和昨日收盘价（一竖指当日股价、两横一竖是均价线和昨日收盘价，两反转结构叫主力动)，再量缩回拉（芝麻量不破两横一竖主力还在）低吸买，且MACD二三次水上交叉（五日战法的不同表达形式-以阴线高点为线、反包为线上阳线），确定洗盘（日线反包）上攻等三格卖（均价线与股价的距离，MACD鳄鱼嘴或死叉）。
- ②短暂洗盘在前15F完成：破低过高、两横一竖完成洗盘 量在仓先，仓在价先分时图与日线是一样的一样的
- ③低开低走强行出货：均价线反压（观察均价线下杀角度，若股价下杀时均价线是走平的跌不动，一定会反抽），拉拐（缩反）就要卖，MADC一波黑绿柱越长出货力度越大
- 3、平开（无缺口）：注意第一波的高点（第一高点）或低点形成自然箱体，等上下翻箱体，①平开高走启动等第一高点被突破的穿（第二高点）座（确认买点）起，②平开低走反弹空（开盘价与第一波低点形成箱体，开盘价为确认点）
### 四、T+0战法：核心是利用一天的波动在所持股中高抛低吸实现当天的盈利，日内交易，降低成本，看涨做短。根据开盘定式买卖，只有盘中急跌急拉(3%）才有套现（三格）高抛低吸机会，选股一定要在龙头股中。

- 在开盘定式看涨（分时图两低一高、破低过高，低开低走要两横一竖）的基础是先买后卖为正T(选日线上升趋势即使当日没有急拉也会创新高是双保险，平台整理的关注颈线的获利了结)。先卖后买为倒T（只在短波段调整中做），
- 如果是建仓一般用三周期共振选股票（持股方法当股价在五日线上走五根阳线后必须每天收盘创新高否则走人），五线战法上车，再会用到分时的买卖点。

### 极简转折模型：所有的交易要有规则，有时间的把握，根据市场的变化而变化不同时间运用不同的模型且先选板块再选模型，买点关键日线的突破全部在分时图上的确认（线上阴线）

- 模型有N种，买入只有两种：线上阴线买和盘中阴线买（下影线都是线上阴线叫盘中阴线、均价线、开盘价）
- 1、 底部确立，涨停模式-集合竞价打涨停战法（从集合竞价涨速选跳空股）（牛棚线上的强势底分型）
- ①股价向下杀跌挖坑，踩扣底线W1W2且水平或向上（贴着牛棚或扣底线）
- ②调整一段时间形成平底（双重或三重）
- ③某日集合竞价高开高走脱离底部（过左高）往往当日涨停（脉冲分时图：没改变原趋势的跳空须要有脉冲分时图，跳空改变原趋势最佳）
- 扣底线模型是跌下来的双底，反弹U型结构（弱势行情中+抄底线走平时蜻蜓点水二次探底买点，线上阴线买，抄底线下拐止损在线下阳线卖）
- 牛棚线模型是上涨中的双底，主升N字结构（指数行情好时用+连扳家数多+核心板块中使用）牛棚三法
- 股价拉升时腾飞线向下压头，一旦其走平就是短期头部。腾飞线改变方向才能买不要走在主力前面，胜率最高的是腾飞线和牛棚线同时向上，抄底线也向上更好。（三线同步发散上升趋势）
- 牛棚三法： 第一法先定位：牛棚线与腾飞线越近容易拉升+抄底线往上，起飞点在股价放量突破腾飞线、市场的热点板块、资金追逐对象。
- 第二法再定调：情绪指标RSI（3.6.9）三线>80形成彩虹桥股价拉升。
- 第三法是定势：有无拉升意图看牛棚线与布林共振点向上突破，股价没在牛棚线上即使BOLL20突破上轨也无效

- ①调整幅度：双针最高价、实体最高点、实体最低价
- ②调整一天（三五线尚未死叉）的回踩5MA只买在上穿3MA（腾飞线）时
- ③调3-6天三五线死叉等金叉踩5MA盘中上穿3MA买入A发力延续涨停

- 是否能涨停看“穿”基准的力度（涨停分时图）
- 期货分时图战法
- ①粘合单边：价格线与均价线水平粘合波动（时间>2.5小时，高度越小越好，日均波幅的1/4）一旦出现强势的突破或跌破（突破粘合最高点且突破的角度越陡越好）信号，极大的概率走出日内单边大行情；（在回踩拐点追）
- ②一侧单边：价格线在均价线上方强势上涨（>日均波幅的1/2，形态越强反复越小)后小角度缓慢向下回踩均价线（幅度<1/2）；一旦向上突破极大的概率走出日内单边大行情；(突破调整趋势线就是追多点)（下跌亦然）；一侧单边的第二种形态回踩均价线不破直接反弹
- ③横盘跟单：价格线在均价线上方以较大的角度上涨（初始上涨幅度>日均波幅的1/3）之后在均价线上高位窄幅横盘震荡（时间>1小时，幅度<初始上涨1/4），极大的概率延续上涨趋势（下跌亦然）；L型跟单:快速下跌(高角度且幅度>日均波幅的1/4，横盘时间>半小时）
- ④垂直追单：价格线围绕均价线小幅粘合波动（波幅<日均波幅的1/4)，垂直拉升调整后极大概率会有第二波上涨（垂直拉升可直接追买，回调可补仓）;反向跳空，价格线在均价线上一波趋势上涨中反向垂直向下（追空），极大概率开展一波向下反转行情
- ⑤反转二波：价格线在均价线下方一波下跌趋势行情，出现连续大角度大比例日内反转，调整之后有第二波上涨（回踩找拐点做第二波）
- ⑥双顶逆转：价格线在均价线上方快速走出双顶形态，注意右边一波跌破颈线（追空点）将开启一波反转向下的走势
- ⑦头肩逆转：价格线在均价线上方快速走出头尖肩形态，跌破颈线追空点；第二种只有右肩的也是跌破颈线追空点
- ⑧尖顶逆转：价格线在均价线上方突然迅速冲高迅速回落，尖顶（底）一旦形成开启反转下跌（成功率非常高的反转形态）在调整中找拐点反向交易
- ⑨三浪逆转：价格线在均价线上方波动后大幅回调再次向上波动再深度回调吞没破三浪起点，反弹无力极大概率行情开启反转向下，在三浪的起点追空反抽不过补仓
- ⑩V型逆转：均价线走在均价线下方的下跌趋势中，突然出现快速下跌又快速上涨形成V结构，开启向上逆转（v型调整确认进场）；开盘V型
