# 8..衔接

- 衔接技巧-找本年度启动票，每月底收盘选一次（模式选股法-月线收盘晚上选一次)
- 一、趋势结构（选股模型）
- 1、空转多（4MA5MA10MA月线-机构中长期运行成本）：股价跌破成本线，连续运行在成本线之下，突然出现一阳穿三线（过末跌高），之后出现均线多头排列
- 2、一阳穿三线（拐点突破旗杆B点）后找空翻多结构（两低一高和破低过高）起点A，若一波流（>二根阳线）见顶（5MA月线上穿10MA月线没有阴线一波年线见顶）
- 3、一阳穿三线后的涨幅测量：A点箱体低与拐点突破箱体高为一个自然箱体向上翻箱体（1、3、5、7、9）测高（以一箱体为半径画圆的外围是时间窗口），反弹第一目标是一箱体,
- ①若弱的<1.618，打不到箱顶会回踩启动阳线1/2(最低点无条件止损)，强的回踩在10MA月线
- ②箱体运行总是奇数箱体拉回不破阳线的1/2继续看涨，一般的个股涨三箱体，如只涨到两箱体就哪来回哪去（进二回一）！
- 二、形态要点
- 1、穿坐起：股价一阳穿三线后形成旗面整理，整理区间均线保持多头排列（阳上阳后面的机会更能够把握一年的行情。其次假阴真阳，再次阴线）
- ①一阳穿三线后阳上阳最好，代表上升趋势确认，假阴真阳算阴线上涨不干脆（两阳夹一阴多方炮也能涨）
- ②阳上阳结构的上影线越长越弱（月线级别里不允许有长上影线，特别是启动后的结构-抛压重上涨难）（长上影线的标准实体的1.5倍至少调整5个月。横盘中均线无效只看箱体，上涨角度>45°只看均线）
- ③阳上阳的启动>30%涨幅大时，正乖率大须回踩5MA月线
- 2、上穿三不破
- ①不破启动阳线的1/2
- ②收盘不破5MA月线（强势回调）破了等踩10MA月线低吸
- ③收盘不破10MA月线（弱势回调）
- 三、衔接要点（一阳穿三线过左高最好）
- 1、调整幅度：股价一阳穿三线后形成旗面整理，整理区间下调保持在5MA月线附近（阳上阳结构调整的时间短，且所有一阳穿三线后不过高的都是当年的顶，跌无底所以阳上阳不能抄底）
- 2、衔接点（只找阴线的高点）：股价一阳穿三线后形成旗面整理踩刹车（第一根阴线高点），衔接点就在阴线的最高价当月衔接或下月下影线衔接-安全（“月线顶底互换”阴线是N型的开始，阴线最高价设预警，长线选股短线交易，完美先大后小）
- 3、绿色弱衔接信号与红色强衔接信号的区别在MACD水下水上。（弱衔接不是主升结构，涨完了要回的）
- 4、月线的衔接点在日线中使用时还要验证，衔接时找日线结构的须一阳穿三线（调整结束信号）+衔接后日线股价占在5MA之上5MA10MA须多头结构三五通道通气状态若4M5AMA死叉全清，等待下一次的一阳穿三线机会
- 四、衔接过渡（三种过渡重点区分一元结构和二元结构，优中选优过渡时间的把握）
- 1、过渡幅度：股价突破旗面整理，K线形式不宜出现长上影线的K线（除极开盘在简转折灯塔线的位置），当月涨幅不宜过大，影响后期的上涨即过渡时间长3-5个月-二元结构-下上下过渡形式。（不要贻误战机在模式中优质选优-有一元结构就不找二元结构的）
- 2、过渡阴线区分：高开阴的假阴真阳过渡看实体最低价（收盘价、最强衔接），真阴线的要过最高价。
- 3、过渡阳线区分：连续连阳一波流后面的阴线一概不要。但高手做阳线站上最高实体（收盘价）
- 五、指标解释（辅助作用并非交易提示）
- 1、月线看粉色数字<5内完成衔接（最好的是3），粉色数字是九转序列。上车后日月看黄色数字13都是阶段见顶（若黄色数字>13时以收盘价止损会多看两天）。
- 2、选股器是用月线选股，画出衔接点（点右键设置条件预警），盘中定时筛选在日线一阳穿三线与衔接点重合处上车（或30F级别衔接）
