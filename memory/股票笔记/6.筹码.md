# 6.筹码

- 主升异动试盘拉升前兆及移动筹码
- 一、长上影线异动试盘（600550、000536、000627）
- ①长上影线异动手法：股价当日在盘中放大单分时冲击波，造成短时间内股价被大资金拉升，当拉升到一定幅度后，盘中买盘资金撤出，挂出卖盘形成抛压状态，导致当日股价冲高回落形成的上影线，试盘结束的表现为整理一段时间后放量突破试盘的最高价。后市将展开拉升模式。
- 个股在盘中分时图上冲高回落，当日试盘结束后进入连续的洗盘震荡调整，随后放量突破当日前期上影线上方，意味着分时图主升模式已经开启。
- ②上影线主力试盘要放量（“一放”即在整理中的最大量，然后“二缩三突破”三步走，整理洗盘动作明显）（上影线出现在短期回调位，也是阶段性放量的地方，试盘K线要在旗杆之后产生，最少要五根日K线、并且上峰要消失）
- ③突破上影线（不分阴阳）当日的启动要出现单峰密集且无上峰，（移动筹码90%成本集中度高了就会出现勺子柄-买点在120FKDJ，否则会直接拉升）300287
- ④两个买点：突破试盘线加仓买二；买一当单峰密集的获利盘>套牢盘,且移动筹码90%成本集中<12%越小越好时可以提前试错买入不用等突破试盘K线605218买点筹码90%成本集9.7%，平均成本17.44元
- 二、长下影线异动试盘（下影线的作用，先洗后拉）
- ①下影线是由于主力为了识别个股在市场中的关注度，有意识在盘中向下砸盘，到股价产生技术支撑位后，在卖盘的位置将卖单撤出，以此来观察相关位置的大资金与散户的买入单量变化，进而判断个股在市场中的人气并且也可通过此种手法达到洗盘的目的，造成部分投资者为了止损而恐慌性抛出低价筹码，从而继续吸筹。此手法一举两得在向下试盘的过程中，只要收回关键位置支撑位也就意味试盘结束。
- ②个股接连收出长下影线，主力同时加大了吸筹的力度，主力在完成吸筹目的后，在伴随放量的状态下，开启主升模式。(至少两个下影线才能判断主力的洗盘002261)
- ③找旗杆的关键是向上拐点。下影线的位置不可跌破上升旗杆的最低价-在上升旗面的才是试盘拉升结构，而下降旗杆中的旗面都是反弹结构000158、002786、002693
- ④第一个下影线最高实体被有效突破试盘结束启动
- 三、底部涨停和多次涨停异动试盘 -
- ①主力为了测试并判断市场筹码的稳定性，通过涨停板观察盘中抛盘情况与跟风盘的资金量情况的产生对比（测试市场抛压），从而进一步来分析筹码的稳定性，在涨停板的第三天或者三天后，股价开始下跌，如果没有下跌，股价也会产生横盘整理状态。
- ②主升启动之前，每一次涨停板过后都会产生迅速的回调，经过一次或多次涨停板试盘之后，股价才会正式启动主升行情002717
- ③在拉升时先单峰密集，再续峰抬高，再次单峰密集后才会拉升启动主升行情（单峰密集+成本发散上行延续+上涨多峰密集续涨）
- ④当涨停后回调的幅度减弱（形成筹码续峰结构，最终形成单峰密集拉升），说明将要启动（001236弘业期货）
- 四、打压挖坑洗盘异动试盘（突破60线后一般不会再破，56MA是散户心理线，洗盘跌破56MA的是恶庄）
- ①主力挖坑试盘主要是有两个目的，第一个目的是为了测试下方买盘的力量承接强度，第二个目的是为了洗出不稳定筹码，进而低位吸筹，从而使用大单猛烈下行打压砸盘的动作
- ②股价在回调后站稳56MA（均线须向上）后，遭遇猛烈向下打压，迅速跌破56MA，伴随成交量大幅萎缩之后以逐步放量的形式再次站上56MA，在底部形成一个碗口形状的坑（黄金坑），此手法结束后，股价开启主升模式
- ③主力建仓完毕，拉升前的洗盘动作，挖坑时间越短越好，试盘的成功率越高
- ④向下打压要有成交量，挖坑的特点在56线附近一定是左大阴，右大阳-须阳盖阴-量在价先-（用结构试盘，影线是用价格试盘）
- ⑤买入基准点在右大阳，最低价试错低吸，最高价突破加仓，过高买第三笔。
- 五、实战技巧
- 1、实战技巧一-上峰不死熊市不止(反弹和反转的区别只有筹码能锚定)
- 在下跌行情中，如果上密集峰没有被充分消耗，并在低位形成新的单峰密集，将不会有新一轮行情的产生。本技巧的实战要领：
- ①上攻行情的充分条件是股价的上方，没有大量的套牢筹码
- ②在低位形成新的峰密集意味着止跌企稳（上面可以有一点，随着横盘的时间慢慢全部消耗掉）
- ③下跌多峰中的每一个上峰都是强阻力位
- ④对于下跌多峰的股票不易草率建仓（不会有新的反转行情只是反弹）
- 2、实战技巧二-突破低位单峰密集
- ①上峰消失新的筹码堆积，形成低位单峰密集 .
- ②黄色获利盘>白色套牢盘（调整时间由筹码决定）（只看日线其周期都是高抛低吸用的）
- ③移动筹码90%成本集中<12%越小越好(用以衡量密集程度，越小越集中，否则隔天易开阴线，因为密集度不高还会继续洗盘）
- ④低位单峰密集启动后的第一个压力位在启动移动筹码90%成本集中度翻一倍的地方
- 3、实战技巧三-上涨多峰密集续涨
- ①单峰密集启动回调变双峰，会洗盘回归单峰密集（单峰变双峰再变单峰说明主力也在高抛低吸，我们只要管住单峰介入双峰不介入)
- ②整理合并填谷再成单峰密集（二元结构整理时间长）
- ③出现的续峰<底峰且获利盘>套牢盘才能跟进
- ④续峰结构是一元结构整理时间短，是良心庄家能带你飞
- 4、实战技巧四-突破高位单峰密集
- 5、实战技巧五-V型反转之峰密集（用在左侧交易，最高技术就是左侧交易和超极短线，左侧交易九次错只要一次对，超极短线套利要快核心减少回撤-集合竞价开盘定式天衣无缝，位置要求很高即择时）
- 股价跌穿单峰密集超跌下行，如果股价超跌原单峰密集并没有被消耗，行情将发生V型反转，反转第一目标位在原单峰密集处。
- ①股价跌穿原单峰密集
- ②股价下跌的速度较快（跌得越快是空跌，反弹越快）
- ③下跌至超跌区原单峰多密集完好存在
- ④超跌V型反转确认后及时跟进
- ⑤股价反弹至原单峰密集区域出场观望（即蓝线平均成本套牢盘位置）
- 6、实战技巧六-成本发散上行延续600550、000536
- 股价突破低位单峰密集上行，成本发散意味着趋势的开始和延续，本技法实战要点：
- ①筹码发散是行情处于展开的状态（实战中上涨中成本抬高的很难出现直接负反馈，上涨中成本抬高很难大跌，单峰调整时间短，反之成本没有抬高的双峰变单峰震荡时间会长）
- ②低位峰密集后上行发散是介入的时机（大幅换手，没有大幅获利盘，行情易延续）（拉升中的续峰不能超过底封300310、续峰结构是主力拉升信号，也是散户明确跟庄信号）（买点：续峰获利盘>套牢盘）
- ③高位峰密集后的下跌发散，不可介入反弹
- ④成本发散对低位密集峰的消耗越少，行情的延续性越强（续峰结构，低位单峰密集没动，上面出现第一个续峰结构有新资金介入，续峰结构是接力性上涨，单峰变续峰越早买入越好，是牛股最好的形状、000536）（拉升中低位单峰密集不动的是吃独食的庄股，没有合力会大跌出现A杀）
- 7、实战技巧七-洗盘回归单峰密集（不单单在低位，高位中位都能使用）
- 当低位单峰密集形成后，股价跌穿单峰密集，随后又回升向上突破原单峰密集，是新一轮行情的开始。本技法实战要点：
- ①经较长时间整理形成低位单峰密集（单峰密集不等同于调整结束，低位单峰密集最容易吃大套亏大钱）找单峰的最大峰值（即最大换手）画切线作为多空分水岭上多下空
- ②对单峰密集的跌穿是主力凶很的洗盘行为（突破了才叫洗盘，洗盘是马后炮的技术）决解方法就是不参与任何洗盘，跌破回归线的不要
- ③洗盘回调的幅度不会太大，通常在20%左右+（筹码绿线代表旗杆平均成本-最小96根是关键，突破易上涨：上升旗杆找回调的平均成本-光标放最高价，跌破完完，下跌旗杆找反弹的平均成本-光标放最低价，突不破是压力002207）在平均成本上的做“穿座起”之下的做“顶回”
- ④回调时原单峰密集没有减少的迹象，成交量萎缩
- ⑤洗盘回调后回升到原单峰密集（切线），是最好的介入时间（一般出现反包线、底分型）（洗盘结束标致：获利盘>套牢盘）（且还要重新形成新的单峰密集-600272、600611、837821、300746）
- 8、实战技巧八-回调峰密集强支撑
- 当股价放量突破低位单峰密集后不久做回调整理，在低位单峰密集处受到支撑，是一波主升行情的信号，本技法实战要点：
- ①通过长期整理已经形成低位单峰密集（横有多长竖有多高）
- ②有放量突破低位单峰密集的行为
- ③突破后经小幅上涨后做回调整理（回调不能跌停板放大量）最少周线出阴线-消化过高的技术指标所以最少要调整五-八天再周线阳包阴，而行进中的回调不能超过三天否则指标背离了
- ④回调整理在原低位单峰密集处受到支撑
- 9、实战技巧九-做顶洗盘再度密集（最难判断的走势）
- 股票形成第一次低位单峰密集后，经过一小幅上攻行情做顶回落，但在顶部第一次低位密集仍被大量的保留，当股价再次回落至第一次密集处形成第二次单峰密集，第二次的密集与第一次密集相重合，这种行情的出现是一波上攻行情的开始，本技法实战要点：
- ①形成第一次低位单峰密集后开始小幅上攻行情
- ②该小幅上攻行情的涨幅让人感到有做头部的可能
- ③在小幅上攻行情的顶部测试第一次低位单峰密集的状态
- ④在小行情的顶部时低位单峰密集仍大量的存在
- ⑤股价回落到第一次密集处，形成第二次单峰密集，第二次的密集与第一次密集重合成单峰密集，
- ⑥股价放量突破第二次合并的单峰密集时是拉升的信号，操作时可再次突破峰密集跟进601162、300380
- 10、实战技巧十-跌穿高位密集止损
- 当股价经过一轮上攻行情，在高位形成密集峰，若股价跌破高位密集峰是行情终结的信号，应及时出局止损。本技法实战要点：
- ①股价已经有较大的涨幅（启动集中度旗杆末跌高的一倍数是关键点）上峰>下峰出货。下峰>上峰是洗盘（工商银行）
- ②原低位单峰密集被消除，形成高位单峰密集（一线天容易出妖股）
- ③在高位单峰密集处不宜介入
- ④当股价跌破高位单峰密集应及时止损
- 试盘K线选日线，周线需要五天确认，月线线需要30天确认
