## ★

- 一、KDJPLUS指标:
- ①可用于5分钟线，做T；短线最小级别用60分钟；周线未金叉向上时，日线金叉进死叉出，快进快出；周线金叉重注做大波段（>20%）；周线死叉清仓！
- ②20以下发生金叉买，50附近发生死叉要卖一次（回归性），20附近等二次金叉再买入。但是低位金叉需要注意：K和J值张开的扩口要大（差值>10），才代表股价短周期上升力道大，如果扩口太小，则金叉无效，不构成买点。
- ③放量（5-10日均量线）的金叉有效、一波上涨后逃顶：指标在80以上高角度死叉、K<80、jazb2指标在250以上
- ④KDJ最大的作用在月周级别，可做长波段；KDJ月周金叉时可抱一波，最好诠释，放量（5日均量）第一根长阳的KDJ金叉叫同步金叉，两根阳线之后的金叉就是非同步金叉；死叉只看位置。
- KDJPLUS操作策略运用
- ①当金叉出现，如果前三天（含金叉当天）没有进场则放弃等另一次金叉再进场
- ②高档钝化（K，D上80）此刻走势最强，一遇钝化金叉，要敢于进场或加码（若迟了两天请耐心等另一次金叉），钝化会产生股价飞速上涨，等到周KD死叉再退场
- ③当KDJ金叉后三条线线平行向上，若做T只能先买后卖（正T），当KDJ死叉后，若没出场则做T只能先卖后买（反T）。
- ④当股价上涨角度是以往没出现过的大角度上涨（加速上涨）那当KDJ一出现死叉K先跌破80时，必须要先清一次仓位。
- KDJPLUS针对J值的操作细则
- ①指标金叉死叉是看不到力道的，但加上J值可以清晰看到力道，例如高角度死叉与高角度金叉
- ②当进行多方时，股价上扬而J值离K、D还远，其实不必担心短线下跌；而当股价开始下跌，J值高角度死叉，当J值离KD还很远，我根本不会考虑去接手股票。当没太多时间复盘，只要简单看一下KDJ状况就知道要怎样看盘，尤其J的位置
- ③当金叉出现，J值还很远，事实上是可以忍受股价整理，但是有一天，当J值快靠近KD时，股价一直涨不动，那就要注意，隔天多空必定表态，尤其在商品期货多空交战时，隔天就很重要
- ④当死叉出现，股价却没跌很多，盘个一两天就停住，那就要注意J值是不是靠近，若靠近，股价不跌，那极有可能第2天会出阳线，然后变成金叉
- ⑤每当看到死叉或金叉，接下来我最重视J值，因为可以让我知道，股价何时快逆转，也就是开始准备反向操作
- ⑥当KD已经拉很高，我不期待一定会钝化，尽量不追高，要等到钝化再金叉，当KD已经掉很低，不能再追空，除非钝化再死叉

- 2.进入上升后KD一定保持向上，此时看J值若与KD有距离，不用怕整理震荡，可不走

- 6.上升也是看J值离KD有多远

- KDJ钝化概念---股价在上47AM均线上易钝化（KD在20以下钝化时也要一底比一底高的情况，钝化无关次数。钝化防的是回归性）
- 1.如果股价还在创新高，J值靠近K、D，不用太担心，因为有机会形成钝化。
- 2.如果股价一直过不了新高，而J值在靠近K、D，那需小心，应该快反转。
- 3.如果股价一直在破底，即使J靠近K、D也没用，因为会低档钝化。
- 4.如果股价不再破新低，而K、D一直下降，J也快靠近了，则这是在冷却指标，一遇到出量的有效金叉，上涨的确认度很高
- 高阶用法
- ①KDJ同步与非同步：正常的金叉、死叉在KDJPLUS的参数与数学逻辑设计下，最慢两根K棒就会出现。若拖到第三天（第三根K棒）才出现金叉、死叉，才称为“非同步现象”(备注：两根K线的涨跌幅<2%时可合并) ，此情况出现，往往反转力道不会太强。反而会走反向
- ②钝化回归（只用日线） 这是KDJ分析上的重大发现。如果在高档曾经钝化过，等未来股价下跌修复，而在低档出现金叉后，往往kdj指标会重新回到80以上，也就是再碰一次高档（点到80为回归结束，如遇重大利空至少点到50）。反之若在低档钝化，则未来反弹后再度下跌时，指标会再重回20以下一次（点到20为回归结束）。换句话说，回归现象是告诉你指标一金叉，就知道可以走到哪里。（指标回归时股价可能创高破低）
- ③盘中切入方式 最佳是日线金叉，而60分钟，5分钟，在低档金叉为最佳。若盘中交战时，“尽量避免在5分钟KDJ高档时切入。做强势股的第二买点最佳方式 是用60分钟来看，也就是日线没问题，而60分钟重回50或20附近金叉，再切入为最好方式。
- ④中级反弹 若月线KDJPLUS未金叉，而周线金叉，少则2个月，多则3个月（50以上金叉），有机会反弹。
- （1）周线金叉，日线死叉，这是小修正
- （2）周线金叉，日线金叉，波段还在进行 （三周）
- （3）周线死叉，日线金叉，视为反弹
- （4）周线死叉，日线死叉，下跌持续
- ⑤长空结束前的征兆 下跌中必有反弹，假设一大段下跌，包括大盘或个股，在反弹过程中，一次比一次指标都走的更高才反转，则代表空头快结束。
- 例如其中一次反弹， KDJPLUS不到50就死叉，下一次冲过50才死叉，再下一次冲过80才死叉，此刻就要看周线，月线指标是否快低档金叉，因为再一次金叉，可能就开始转为 中长期多头格局。
- KDJPLUS最难用法：时间的推演
- （1）用在早盘结束后的推算
- （2）用15KDJ来看，如果早盘收盘15KDJ是低档金叉，则下午势必再攻。反之若高档死叉，下午先走低
- （3）若已经金叉再进行，或者死叉在进行。（可看J与K、D的距离）可用沙盘推演去看未来几根K棒变化，一根是15分钟。如此可看到未来多久会由向上转为向下，或向下转为向上。
- （4）当推估时间有了基本概念后，则可看个股的变化 。例如看出来几分钟是先下后上，则在下的过程中， 一直抗跌的股可留意。而时间快到（再配合POWER指标）时，此股先行向上，则便知是强势个股。
- （5）时间推算要靠经验，要长期（短则3个月）训练 ，不太可能短时间练成功力。

- ②KDJ从低档金叉直接到高档第一次死叉，离场讯号；

- 二、五轨BOLLPLUS
- 1、大盘和个股皆可用，周期从5、15、30、60、日、周线皆可用，但月线周期太长不适用
- 2、股价由高位往下（无论是一二三轨起跌）崩跌，如果一开始就破五轨是很严重的事，不管用什么指标，都不可以抄底。要直到股价重新收敛在四、五轨之间才可判断初步止跌（在各周期上使用原理相同）
- 3、当股价收敛在四、五轨，出现反弹，而第四轨一直上不去（通常出现在日线，而且在弱势股），则便是弱势反弹。而指标再出现KDJPLUS死叉，会再继续跌，此刻便要看五轨是否守住，若再跌破（收盘前确认），则又是另一段下跌的开始。
- 4、若股价在五轨守住，也上行到四轨之上，基本上都会有机会攻到三轨（中轨），此处压力大若三轨也过，就是一关一关看，到二轨，到一轨，而到一轨，无论如何要走一趟。
- 5、当股价由弱转强，会从一轨拉回后守二轨若二轨一直不破，再上攻时突破一轨，则攻势会更猛烈（大多头格局中才能用）
- 6、在上攻中超强势股，会一直守二轨。也就是二轨一定有支撑，直到有一天拉回二轨，而再上行已经碰不到一轨，则K线一收黑，则为卖讯（日线到短周期皆可使用）
- 7、这是抓强势股起涨前的用法：
- ①如果股价走横向，而股价一直在二、三轨之间或一、二之间游走，则耐心等待，
- ②若出现一根带量长红，便是强势上涨的开始（高倍反观高温可忽略、站一轨之上追买）。往往这种股，在上攻中几乎都会在一轨之上，而整理不会破二轨（破二轨又破颈线危险），直到有一天收敛在一二轨间，碰不到一轨才会休息。
- 8、真正强势波动的格局表现：
- 日线周线位置都在一、二轨间盘整，也就是已上涨一大段，再进行整理时，出现此现象，代表后头还会有一段走势，这种格局股票要多留意（如果一、二轨道扩口不断放大，就很难防守）
- 9、看大盘的技巧：
- ①短周期波动中5、15、30、60分，每五轨位置都是压力、支撑价，也就是关键价，在分析上以15分钟为主，若看支撑、压力的位置太远或太靠近，可转换不同周期去研判（此技巧要长期训练，多看会有概念）
- ②短波操作：一轨常是压力，贴一二轨攻击、破二轨小心、守五轨收敛，四轨不过再下、过四轨反弹开始或向上
- ③均线参数和指标参数设置对时，会出现不同周期都到防守区--共振强支撑（60分钟MA71线支撑与守四轨15分钟在五轨、布林走平KDJ金叉买入）
- 10、极短线做T：若波动温和、从5分钟看，若一轨过不了，可先出，直到五轨才收回，这是短T的方式（赚2-3%）
- 11、强势上涨的停顿：股价由一轨之上收敛在一二轨之间，可先走，然后再看二轨的防守状况（一轨之上收敛回来要多留意，远离三轨要小心了，三轨不过马上见长阴，五轨走平企稳才能买、）
- 12、强势下跌的停顿：当股价在五轨之下都不是超跌，在空头行情的起跌段不测底，千万不要碰。（除非末跌段负乖离过大）
- 13、五轨会收敛也会变大。这没有任何迹象显示会往哪边走，重点技巧：若五轨开始上弯，则底部支撑会强一些，反之若在高档盘整过程中，若一轨开始下弯，这会是极大的压力，以大概率而言，多方气势快结束了
- 14、再度强调：从五轨上升到一轨，尤其是大盘，先都要走一趟，走完后拉回再看防守位置。

- 三、高倍反向观测指标（jazb2）：----抄底逃顶杀器
- 用于日线级别可以在任何指标都还没有死叉前提前发出警告，在涨势中,分急涨与缓涨：
- 缓涨的： ①大都以大票为主（市值>100亿，越大越好），或一些成长型股票，如果低于-150，等待二次金叉入场；零轴上方的金叉、股价在相对低位发出买入讯号；
- ②当高倍反观指标慢慢上升而快到250左右，那就是警讯，如果此刻MACD柱子缩短或机构托盘指标有点上不去，布林线破二轨了，那再去接手就是追高，死叉离场；
- ③如果高倍反观指标超过300必须立刻离场。
- 急涨型： ①一般都是炒概念炒筹码炒投机，这些票一开涨，指标都会很高，所以250是不受限制，这因为与扣底与原始轨道变化有关，未来回落指标会守100再来BOLLPLUS会守2轨或3轨，金叉可再二次攻击、守0再来三次攻击；如果特别强势的贴5MA上攻指标回150金叉
- ②股价过高指标不一定过高，此指标没有背离，且指标极限会慢慢下降。
- 1.当指标重返-100以上进场（负乖离过大买点，配合KDJ金叉捡带血筹码），原则上都有机会向上走一段，但到0轴就要小心压力出现（KDJ第一次死叉走一次、过0轴才算过压力）。
- 2.很多指标无法解释的现象：当股价一起涨，所有指标都行，但为何会失败，原因在高倍反观指标里可以找到，如果指标在股价一起涨就冲过150以上甚至更高，通常都会压回，甚至又开始盘整 ，这点在一般看金叉操作的人是无法理解的。
- 3.正常的起涨，要在-100以上，指标金叉，然后慢慢爬升，到0轴是压力（多头格局压力小）。若0轴站稳，接下来攻击会更猛，下一个关卡直接看100，过了100力量会更强，但相对的股价会来到高位，如何判断，看（4）（股价在涨指标在回档是该指标的技术背离-退场）。
- 4.当股价过100，下一个是150，此处压力不算太大，但接下来几个位置很重要，也就是公式里写的，200、250 、300、350、400，也就是每超过50都要小心其他指标变化，或盘中波动， 其中250以上都算危险区。
- 5.每当股价由上回落，往往到100会有支撑，若100支撑守住，股价再往上攻，第二次上攻，即使股价过新高，在高倍反观指标上不一定过高，这不是背离，是软件设计上的问题(均线扣抵)。若出现此现象，需要使用其他指标（KD,BOLL）来辅助逃顶。
- 6.若股价回档较强到0，则反弹就要特别留意到100的压力，若100已攻不上，代表多方走势已经结束。接下来破 0的机会很大，要小心，都有可能跌到 -100以下。
- 7.若是大多头走法，在每遇回档中，第一关会守100，第二关守0，若0一直不破有机会在往上攻一波，但会攻到何处，就前所述是不容易再过前一波指标高点。
