# 4、神奇数字

- 一、交易的关键点：技术分析中形态是首要的（形态由成交量来确认）
- （一）、趋势结构：E点结构（E点是要从前期热点板块中精选调整下来符合模型的品种）。破底过高和两低一高两个做底结构。第三个底就叫E点结构（一创新高D点买易吃套，只有10%“春笋”型的不在模式内，严谨做D点）
- 了解开盘定式，发现主力意图（下降趋势、横盘整理、上升趋势）。要快只能在上升结构中，真正赚钱的只有两种模式（破低过高和两低一高）
- 1、破低过高模式（模型是静态须叠加动态市场热点）
- ①三重底结构，(穿)CD>AB（顶）,DE>BC（回）、颈线B之上的E（坐）最强-形态结构支撑之上最好
- ②D点形成空翻多结构（短线关注的是游资拉升票，重点在CD位置必须空翻多结构，空翻多游资短线进场）但CD角度<60°不能释放得太多（要在密集区越近买入越好）
- ③E点股价回调均线跌而不破长期均线，回拉找买点（炮坑洗盘选股、E点最低价计算低吸点和F点启动加仓点）
- ④F点（起）启动均线长短同步向上，股价形成上穿突破
- 2、两低一高模式（过高回拉找买点）AB须是空翻多结构（二次交叉结构）
- ①双底结构（两底的高低差在5%-7%），低点抬高C>A（量能一放二缩三突破即D点突破）（双底的C点是确认底部的过程容错率差失败会创新低成中继，E点是确认延续趋势问题失败概率小只是时间问题，调整是一元或二元或三元结构到箱体震荡）
- ②过高D点回拉E点在旗杆的1/2上为强（五大结构+均线最大功效弱转强+先大后小）（E点是三浪三结构主升段，抓得住的是常态-操作，抓不住的是诱惑-欣赏）
- ③回调二元结构，回调时间越短越强，回调不要破乾线的根即最低价
- ④F点启动（过D高）会出现连阳拉升，主升开始（上涨常态只做T,不加仓，用三五通道持股）
- 3、轨道线：轨道线买卖法则（适合比较陡峭的趋势通道）①上升趋势：一买三卖法则、②买入：价格回落到趋势线时（E点）、③卖：价格突破趋势线，价格抵达轨道线、价格向上突破轨道线
- （二）、启动确认：反转信号、向上缺口
- 1、 阳线的收盘价突破前期高点为有效突破（上影线无效）
- 2、 收盘价突破后运行三个交易日出现回调，不破前期高点的收盘价，为突破有效
- 3、放量突破下降趋势线，回拉确认点也是E点F点（E点过左高）
- 4、形态识别：上升三角形突破D点是主力试盘动作，禁止上车点

- （三）、强弱判断：力度识别、市场承接（观察E点的承接-形态的低点是最主要的防线，不创新低了，看跌速和涨速-大长腿）
- 1、市场上的承接千变万化，承接的好坏，是一切操作的标准，，买入看承接，卖出看承接，承接强就买，承接弱就卖。
- 2、承接盘是指在某一价位附近不断的有资金买入的买盘，一般承接盘都是给上户挂的，成交了才是真实，分时就看看大单成交个数就行，主要看一天的量大小
- 3、承接其实就是当前买卖盘的变化，潜在买盘跟潜在卖盘的变化；承接好几种，分为集合竞价承接、开盘承接、盘中承接、日K线承接等。
- 开盘承接：集合承接基本上可以看出强与弱，在开盘承接的过程中，就是验证一下强弱的延续，有些个股可能，因为集合竞价过后5分钟思考时间，很多参与者会看整体的情况来判定自己整天的交易。在市场模棱两可的时候需要开盘承接来验证一下集合的真假。
- 开盘承接也是千变万化的，要根据当时的情况，要考虑潜在买盘和潜在卖盘。
- 买盘强势的情况是，价格没变或者价格下降，同时成交量下降，这种情况就是买盘缺少，买盘找不到对手盘，这种极致的情况是价格往下掉，都是成交量少，这种是潜在的买盘少，潜在卖盘多。
