## Apify + Claude Code 网页抓取

- 来源：微信公众号「恶人笔记」- 作者 jackao
- 主题：Apify MCP Server 让 Claude Code 实时抓取全网数据
- 官网：https://apify.com/agentify/claude-code
- MCP 集成指南：https://mcp.harishgarg.com/use/apify/mcp-server/with/claude-code

### 核心能力
- 3000+ 预建 Apify Actors
- 支持：网站、电商、社交媒体、搜索引擎、Google Maps
- 可用工具：6 种

### 配置步骤
1. `npm install -g @apify/mcp-server`
2. 获取 Apify API Token（Apify Console → Settings → Integrations）
3. 在项目 CLAUDE.md 添加 MCP 配置

### 典型场景
- Google Maps 商家信息提取
- Instagram 档案分析
- 电商价格监控（Amazon 等）
- 搜索趋势分析
