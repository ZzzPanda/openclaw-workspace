# Bezi 调研报告

## 什么是 Bezi？

**Bezi** 是一款协作式 3D 设计工具，专注于游戏和应用的快速原型制作。

### 核心特点
- 🖱️ **类似 Figma 的界面** - 2D 设计师也能轻松上手 3D
- 🤝 **实时协作** - 团队多人同时编辑
- 🎮 **AI 辅助** - 自然语言生成 3D 资产、编写脚本
- 📦 **多平台导出** - 支持 Web、移动端、PC
- 🔧 **Unity 集成** - 可导出到 Unity 继续开发

### 融资情况
- 2025年6月 完成 **1300万美元** A轮融资
- 目标： democratize 3D UGC (用户生成内容)

---

## Bezi 核心功能

| 功能 | 说明 |
|------|------|
| 3D 建模 | 基础几何体 + 变形工具 |
| 动画 | 关键帧 + 脚本动画 |
| AI 资产生成 | 文字描述生成 3D 模型 |
| 脚本自动化 | AI 写代码 (基于项目上下文) |
| 协作 | 多人实时编辑 |

## 竞品分析

### 🎯 直接竞品 (3D 设计/协作工具)

| 工具 | 特点 | 定位 |
|------|------|------|
| **Spline** | 浏览器端 3D 设计，AI 生成， 免费开源 | 与 Bezi 直接竞争 |
| **Figma** | 2D 设计巨头，3D 插件生态 | 设计师迁移路径 |
| **SketchUp** | 建筑/室内 3D，简单易用 | 专业建模 |
| **Autodesk Maya/3ds Max** | 专业 3D 制作 | AAA 制作 |

### 🎮 游戏引擎 (AI 辅助)

| 工具 | AI 特点 | 适合 |
|------|---------|------|
| **Unity AI** (2026 GDC) | 文字提示生成完整休闲游戏 | 门槛降低 |
| **Unreal Engine 5** | Blueprints 可视化编程 + MetaHuman | AAA/高端 indie |
| **Godot** | 开源免费，GDScript | 独立开发 |
| **GDevelop** | 无代码 + AI 辅助 | 完全新手 |

### 🌍 元宇宙/UGC 平台

| 平台 | 特点 |
|------|------|
| **The Sandbox** | Voxel 3D 游戏制作器，完全无需编程 |
| **Yahaha** | 低代码元宇宙游戏创建平台 |
| **Core** | 免费 UE 授权，MMO/射击游戏模板 |

### 🤖 AI NPC/角色

| 工具 | 特点 |
|------|------|
| **Inworld AI** | AI NPC 创建，语音对话，记忆系统 |
| **Inworld Studio** | 免费创建智能 NPC |

### 📝 AI 故事/游戏生成

| 工具 | 特点 |
|------|------|
| **Rosebud AI** (lab.rosebud.ai) | AI 生成 RPG 角色、剧情 |
| **RPGGO** | AI 文字冒险游戏生成 |
| **AI Dungeon** | AI 驱动文字冒险 |
| **Ludo.ai** | 游戏研究+设计辅助平台 |

## 市场趋势 (2025-2026)

### 🔥 大事件
1. **Unity AI Beta (GDC 2026)** - 用自然语言提示生成完整休闲游戏
2. **Unreal Engine 5.6** - Nanite/Lumen 成熟，中端硬件也能跑
3. **AI Game Jam 兴起** - 24-48 小时 AI 辅助开发比赛

### 📈 趋势
- **无代码/低代码** 游戏制作爆发
- **AI NPC** 从脚本 → 动态对话 + 记忆
- **浏览器端 3D** 工具崛起 (Spline, Bezi)
- **UGC 平台** 融合 AI 生成
